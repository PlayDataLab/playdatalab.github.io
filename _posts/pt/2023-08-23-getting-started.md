## O que é PlayData?
O PlayData foi desenvolvido como uma modificação do Scratch 3.0, voltado para alunos do ensino fundamental e médio. PlayData procura apoiar os alunos na compreensão dos dados através da criação de representações pessoais, aproveitando o poder e a flexibilidade oferecidos pela linguagem de programação Scratch. 

A ferramenta é composta por (1) novos blocos de programação para importar, analisar e visualizar dados, (2) uma tabela embutida no ambiente que permite ao usuário visualizar o conjunto de dados importado, que é aberto através do ícone de tabela (3), e (4) novos sprites e fundos customizados que podem ser utilizados em projetos de visualização de dados.

![PlayData_Environment](/assets/img/playdata-environment.png)PlayData faz parte de um projeto de pesquisa que se concentra no desenvolvimento de novas abordagens para crianças e jovens interagirem com dados.

#### Blocos PlayData: 
Os blocos PlayData foram projetados para executar três funções principais: (a) importar conjuntos de dados (usando um link de uma planilha on-line, copiando e colando colunas de uma planilha ou adicionando manualmente valores a uma lista); (b) alterar a escala dos valores do conjunto de dados para que possam ser usados ​​para criar diferentes representações no ambiente PlayData; e (c) iterar sobre os valores no conjunto de dados para lê-los. Ao combinar esses blocos com as primitivas originais do Scratch, os usuários podem criar representações muito diversas (como animações, músicas e obras de arte, bem como gráficos tradicionais) com scripts que contêm um pequeno número de blocos.

![PlayData_Environment](/assets/img/sample-code.png)## Como posso usá-lo?
PlayData é uma ferramenta gratuita baseada na web que pode ser usada por qualquer pessoa. 
No momento, os blocos estão traduzidos para inglês, português e japonês. Entre em contato se quiser nos ajudar a traduzir o PlayData para outros idiomas!

## Como posso começar?
Para começar, você pode[open the tool](https://playdatalab.github.io/editor)e use o[resource cards](https://docs.google.com/presentation/d/15Jf-HyuaVomNzCEYm89R8vfnGuoGCJ7yqpuK_OiLx7s/)para criar um projeto inicial. 

Alternativamente, você pode abrir um dos projetos de amostra apresentados em nosso site e brincar com ele!

#### Importando dados

Se quiser que um conjunto de dados comece a brincar com a ferramenta, você pode usar[this one](https://docs.google.com/spreadsheets/d/14EOqQbfeSMmmu06tV0S5A6azaNYIRpjvpAcu7Cq6ugU/edit#gid=0), com a posição, magnitude e profundidade dos terremotos ao longo do século passado (com magnitude > 6).

Sim! Para salvar seus projetos você pode baixá-los para o seu computador selecionando esta opção no menu Arquivo.
Você pode carregar seus projetos salvos no PlayData selecionando a opção “carregar” no menu Arquivo. 

## O que mais devo saber?
PlayData está atualmente em desenvolvimento. Assim, novos blocos e possivelmente modificações em blocos existentes podem ser implementados a qualquer momento.
