Você pode se inspirar para começar a usar o PlayData com estes <a href="https://drive.google.com/file/d/10FSxLJpKXuCVbv1RLwqqcgvoZzFac6FP/view?usp=drive_link" target="_blank"> cartões de recursos</a>, que mostram como criar alguns projetos simples com PlayData. 

Ao usar os cartões, experimente alterar os parâmetros e adicionar novos blocos para ver os efeitos!

Não se esqueça de **importar dados para seu projeto** antes de criar visualizações. Você pode usar essa <a href="https://docs.google.com/spreadsheets/d/14EOqQbfeSMmmu06tV0S5A6azaNYIRpjvpAcu7Cq6ugU/edit#gid=0" target="_blank">base de dados</a> para começar a experimentar.
