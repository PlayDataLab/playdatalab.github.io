## O que é o PlayData?
O PlayData foi desenvolvido como uma modificação do Scratch 3.0, voltado a estudantes da Educação Básica. O PlayData permite a criação de representações pessoais para dados, aproveitando o poder e a flexibilidade oferecidos pela linguagem de programação Scratch.

A ferramenta é composta por (1) novos blocos de programação para importar, analisar e visualizar dados, (2) uma tabela incorporada no ambiente que permite aos usuários ver o conjunto de dados importado, que é aberto através do ícone de tabela (3), e (4) novos atores e cenários personalizados que podem ser usados em projetos de visualização de dados.

![PlayData_Environment](/assets/img/playdata-environment.png)

O PlayData faz parte de um projeto de pesquisa que foca no desenvolvimento de novas abordagens para crianças e jovens se engajarem com dados.

#### Blocos do PlayData:
Os blocos do PlayData foram projetados para realizar três funções principais: (a) importar conjuntos de dados (usando um link de uma planilha online, copiando e colando colunas de uma planilha, ou adicionando valores manualmente a uma lista); (b) alterar a escala dos valores do conjunto de dados para que possam ser usados para criar diferentes representações no ambiente PlayData; e (c) iterar sobre os valores no conjunto de dados para lê-los. Ao combinar esses blocos com os blocos originais do Scratch, os usuários podem criar representações muito diversas (como animações, música e arte, além de gráficos tradicionais) com códigos que contêm um pequeno número de blocos.

![PlayData_Environment](/assets/img/sample-code.png)

## Como posso usar?
O PlayData é uma ferramenta gratuita e online que pode ser usada por qualquer pessoa.
No momento, os blocos estão traduzidos para inglês, português, espanhol e japonês. Entre em contato se você quiser nos ajudar a traduzir o PlayData para outros idiomas!

## Como posso começar?

Para começar, você pode abrir <a href="https://playdatalab.github.io/editor/?project_url=/projects/Earthquakes_sample-en_pt.sb3" target="_blank">esse projeto-exemplo</a>, e experimentar fazer algumas modificações nele.

Se preferir, você pode [abrir o editor ](https://playdatalab.github.io/editor) e usar os <a href="https://drive.google.com/file/d/10FSxLJpKXuCVbv1RLwqqcgvoZzFac6FP/view?usp=drive_link" target="_blank">cartões de recursos</a>, para criar um projeto inicial. Não se esqueça de importar seus dados antes  (veja abaixo como fazer)!


#### Importando os dados

Se você quer um conjunto de dados para começar a experimentar com o PlayData, pode usar [este aqui](https://docs.google.com/spreadsheets/d/14EOqQbfeSMmmu06tV0S5A6azaNYIRpjvpAcu7Cq6ugU/edit#gid=0), com a posição, magnitude e profundidade de terremotos do século passado (com magnitude > 6).

Basta colar o link no bloco **"criar listas do link"** e clicar sobre ele:

## Posso salvar meus projetos?
Sim! Para salvar seus projetos, você pode baixá-los para o seu computador selecionando a opção no menu Arquivo.
Você pode carregar seus projetos salvos no PlayData selecionando a opção "carregar" no menu Arquivo.

## O que mais devo saber?
O PlayData está atualmente em desenvolvimento. Portanto, novos blocos e possivelmente modificações nos blocos existentes podem ser implementados a qualquer momento.
